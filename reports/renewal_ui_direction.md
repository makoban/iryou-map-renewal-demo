# 医療MAP リニューアルUI方向性

## 目的

既存の「地域密着の医療機関データ」を活かしつつ、スマホ利用者が迷わず病院を探せる検索体験へ刷新する。

## UIコンセプト

- 第一画面で「地域」「診療科目」「現在地」「キーワード」を横断検索できる。
- 検索結果はカード型にして、電話・地図・公式サイト・診療時間をすぐ押せる。
- 施設詳細は、住所・電話・診療科目・診療時間・休診日・写真・地図・最終確認日を整理する。
- 掲載希望の医療機関向け導線は、一般利用者の検索導線とは視覚的に分ける。
- 信頼性の表示として「足で確認した地域情報」「最終確認日」「掲載情報の修正依頼」を明示する。

## 画面構成案

1. トップ
   - ヘッダー: 医療MAPロゴ、掲載希望、メニュー
   - 検索パネル: 地域、診療科目、現在地、キーワード
   - クイック導線: 休日診療、歯科、小児科、内科、薬局
   - 地域リンク: 愛知、三重、静岡、岐阜

2. 検索結果
   - 条件表示: 名古屋市千種区 / 内科 / 85件
   - フィルタ: 本日診療、駐車場、夜間、駅近、写真あり
   - 施設カード: 名称、診療科目、住所、診療時間、休診日、距離、電話、地図、公式サイト
   - 地図/リスト切替

3. 施設詳細
   - 施設名、写真、電話、地図、公式サイト
   - 診療時間表
   - 基本情報、アクセス、対応科目
   - 最終確認日、情報修正依頼
